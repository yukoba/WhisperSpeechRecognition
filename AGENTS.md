本プロジェクトは Windows 11 向けの音声文字起こしの常駐アプリです。
- Visual Studio 2026 を使用した .NET 10 の C# の WPF (Windows Presentation Foundation) デスクトップアプリです。
- Windows 11 の WSL2 (Ubuntu 24.04) の中からAIコーディングエージェントの OpenCode を使用しています。`dotnet` コマンドは `dotnet.exe` のように `.exe` 拡張子をつけると動作します。
